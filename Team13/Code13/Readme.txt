Instructions :
To compile the Autotester on SPA : 
Inside Visual Studio, navigate to Build > Rebuild All.

To run the Autotester :
Open the command prompt
Inside the command prompt, navigate to Team13/Code13/build_win/x86-Debug/src/autotester
On the command line, enter the following to run the executable 
AutoTester (file path of source file) (file path of query file) (your desired output file)

Visual Studio version used :
Microsoft Visual Studio Community 2019
Configuration : 
Debug
